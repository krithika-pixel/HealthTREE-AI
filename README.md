  **🌿HealthTREE AI**

 ❤️HealthTREE is an intelligent healthcare assistant designed to help users understand symptoms, access health information, and make better health decisions.

❤️It provides instant responses to health queries, lifestyle guidance, and personalized suggestions through an AI-powered chatbot.

❤️It is available 24/7 and provides a user-friendly interface for accessing health information.
